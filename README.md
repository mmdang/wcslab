# Wireless Communications System Laboratory Website

This is a responsive static website for the Wireless Communications System Laboratory (WCS Lab), designed for GitHub Pages.

## Files

- `index.html`: main website file
- `assets/css/styles.css`: design and responsive layout
- `assets/js/script.js`: mobile menu and footer year

## How to publish on GitHub Pages

1. Create a new GitHub repository, for example `wcslab`.
2. Upload all files and folders in this project.
3. Go to `Settings` → `Pages`.
4. Select `Deploy from a branch`.
5. Choose `main` and `/ (root)`.
6. Click `Save`.
7. Your site will be available at:

```text
https://YOUR_GITHUB_ID.github.io/wcslab/
```

## Items to edit

Open `index.html` and update:

- `your-email@kongju.ac.kr`
- office location
- professor photo or lab photo
- members
- news
- full publication list link
